Detailed Description

FileManagerWidget is a main widget class for displaying filesystem.

It provides basic functionality of a file manager - possibility to change view mode, copy/paste and undo/redo support, possbility to change files and folders sort order, etc.

Also it provides actions for basic file manipulation - creating new folders, removing files, showing file info.
