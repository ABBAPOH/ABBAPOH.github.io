Detailed Description

Main application class. Application object is created in main() function and exec() is called. This object lives as long as application is running and owns different singletone objects - ExtensionSystem::PluginManager and Parts::SettingsPageManager.

Applicaion creates main menu bar and common menus - File, Edit, etc.

Also Application class provides typical slots like about(), showSettings() and so on.
