Member Function Documentation

Separator * ModelContainer::firstSeparator ( ) const

Returns pointer to the separator placed before Commands created with model data.

You can insert actions before this Seprator. To add actions after the model commands, simply insert new commands at the end.

By default, separator is invisible.
